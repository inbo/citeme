# Interactively create a new `org_item`

Interactively create a new `org_item`

## Usage

``` r
new_org_item(languages, licenses)
```

## Arguments

- languages:

  A character vector of language codes.

- licenses:

  A list of license items.

## Value

An `org_item` object.
