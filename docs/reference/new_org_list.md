# Interactively create a new organisation list

An interactive alternative for `org_list$new()`. Reuses available
organisations where possible.

## Usage

``` r
new_org_list(git)
```

## Arguments

- git:

  An optional string with the absolute path to a git organisation. E.g.
  `"https://github.com/inbo"`
