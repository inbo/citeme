# Ask for a language

Ask for a language

## Usage

``` r
ask_language(org, prompt = "Which language?")
```

## Arguments

- org:

  An `org_list` object containing the available languages.

- prompt:

  the string printed when prompting the user for input. Should usually
  end with a space `" "`.
