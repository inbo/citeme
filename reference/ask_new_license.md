# Ask one or more licenses.

Ask one or more licenses.

## Usage

``` r
ask_new_license(licenses, type = c("package", "project", "data"))
```

## Arguments

- licenses:

  A named vector of available licenses. The vector contains the URL to
  the markdown version of the license. Use the abbreviation of the
  license name as names.

- type:

  The type of license. Must be one of `"package"`, `"project"` or
  `"data"`.
