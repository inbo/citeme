# Get the list of available organisations

This function retrieves the list of available organisations from the
local configuration files. It returns a list containing the names,
languages, licenses, ORCID, Zenodo, ROR, website, and logo of the
available organisations.

## Usage

``` r
get_available_organisations()
```

## Value

A list containing the names, languages, licenses, ORCID, Zenodo, ROR,
website, and logo of the available organisations.
