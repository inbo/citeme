# Does a person object has a given role?

Does a person object has a given role?

## Usage

``` r
has_person_role(individual, role)
```

## Arguments

- individual:

  A `person` object.

- role:

  A character vector of roles to check for.

## Value

A logical vector indicating whether the individual has any of the
specified roles.
