# Validate URL

Validate that a URL is a string of length 1 which is not NA and matches
the pattern of a valid URL.

## Usage

``` r
validate_url(url)
```

## Arguments

- url:

  A URL to validate.

## Value

A logical value indicating whether the URL is valid.
